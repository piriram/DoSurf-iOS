## 프로젝트 개요

### 앱 소개
**DoSurf(두섭이)**는 서퍼들이 서핑 기록을 체계적으로 관리하고, 실시간 해양 정보를 통해 최적의 서핑 조건을 찾을 수 있도록 돕는 서핑 전문 앱입니다.

### 핵심 가치 제안
1. **한국 특화**: 한국 주요 서핑 스팟에 특화된 정밀한 해양 정보
2. **초보자 친화**: 복잡한 기상 데이터를 이해하기 쉽게 시각화
3. **개인화**: 나의 서핑 기록을 바탕으로 선호하는 조건 분석
4. **한국어 우선**: 한국어 UI/UX, 영어 지원

### 문제 정의
- 기존 해외 앱(WindFinder 등)은 영어만 지원하고 한국 해변 정보가 부족
- 초보 서퍼는 복잡한 기상 데이터를 이해하기 어려움
- 서핑 기록을 체계적으로 관리할 방법이 없음
- 자신에게 맞는 서핑 조건을 파악하기 어려움

### 솔루션
- 한국 주요 서핑 스팟 30+ 곳 상세 정보 제공
- 직관적인 차트와 아이콘으로 기상 정보 시각화
- 서핑 일지 작성 및 통계 분석 기능
- 개인 선호도 기반 추천 시스템

## 타겟 사용자

### Primary Persona: 초보 서퍼 김서핑 (28세)
- **배경**: 서핑 경력 6개월, 주말마다 양양/부산 방문
- **니즈**:
  - 오늘 파도가 초보자에게 적합한지 알고 싶음
  - 언제 서핑했는지, 그날 파도가 어땠는지 기록하고 싶음
  - 자신에게 맞는 파도 조건 찾기
- **페인포인트**:
  - 영어 앱은 용어가 어려움
  - 기상청 데이터는 전문가용이라 해석이 어려움

### Secondary Persona: 중급 서퍼 박파도 (35세)
- **배경**: 서핑 경력 3년, 월 8회 이상 서핑
- **니즈**:
  - 상세한 파고, 파주기, 풍향 정보
  - 과거 기록과 비교하여 베스트 컨디션 찾기
  - 여러 해변 비교하여 최적 장소 선택
- **페인포인트**:
  - 여러 앱을 오가며 정보 확인하는 번거로움
  - 과거 기록을 정리하기 어려움

## 핵심 기능

### v1.0 기능 (MVP)

#### 1. 해양 예보 차트
**사용자 가치**
- 오늘과 내일의 3시간 단위 해양 정보 확인
- 현재 시간대가 자동으로 화면 상단에 표시
- 파고, 풍향, 수온, 기온을 한눈에 파악

**제공 정보**
- 파도 높이 (m)
- 파도 방향 (°)
- 파주기 (초)
- 풍향 및 풍속
- 수온
- 날씨 (맑음, 흐림, 비 등)
- 기온

**업데이트**
- 6시간마다 자동 업데이트
- 수동 새로고침 가능

#### 2. 서핑 기록 관리

**기록 생성**
- 서핑 시작/종료 버튼으로 간편하게 시간 기록
- 종료 후 바로 평가 화면으로 전환
- 1~5점 평점으로 오늘의 파도 평가
- 자유 메모 작성 (최대 500자)

**기록 항목**
- 서핑 날짜 및 시간
- 해변 위치
- 해당 시간대 해양 데이터 자동 저장
- 개인 평점 (1~5점)
- 코멘트
- 중요 기록 핀 고정

**기록 수정/삭제**
- 날짜 및 시간 수정 (당일 내에만)
- 평점 및 메모 수정
- 스와이프로 간편 삭제

#### 3. 기록 조회 및 필터

**필터 옵션**
- 날짜 범위 선택
- 해변별 조회
- 평점별 조회
- 핀 고정 기록만 보기

**정렬 옵션**
- 최신순/과거순
- 평점 높은순/낮은순
- 해변별 그룹화

**상세 보기**
- 기록 상세 정보
- 해당 시간대 차트 데이터
- 메모 전체보기

#### 4. 통계 및 분석

**카드 형식 통계** (좌우 스와이프)
- **선호 해변 분석**
  - 가장 많이 방문한 해변 Top 3
  - 해변별 방문 횟수 차트

- **평점 분석**
  - 평균 평점
  - 월별 평점 추이
  - 5점 받은 날의 공통 조건

- **서핑 활동**
  - 총 서핑 횟수
  - 총 서핑 시간
  - 월별 활동 그래프

- **최근 기록**
  - 최근 7일 기록 요약
  - 최근 방문 해변

**고정 차트**
- 사용자가 자주 보는 통계를 상단에 고정

#### 5. 해변 선택

**지역 분류**
- 강원도 (양양/강릉/속초 등)
- 제주도
- 부산/경남
- 포항/울산
- 서해/남해

**해변 정보**
- 해변명
- 지역
- 위도/경도
- 특이사항 메모

**기능**
- 지역별 해변 목록
- 해변 검색
- 즐겨찾기 설정
- 최근 본 해변

#### 6. 다국어 지원
- 한국어 (기본)
- 영어
- 시스템 언어 자동 감지

## 화면 설계 및 사용자 플로우

### 1. 메인 화면 (홈)

#### 구성
```
┌─────────────────────────┐
│  ≡  📍 양양 죽도  ▼  ⓘ  │  ← 해변 선택
├─────────────────────────┤
│                         │
│   📊 통계 카드 영역      │  ← 좌우 스와이프
│   ○ ● ○                │  ← 페이지 인디케이터
│                         │
├─────────────────────────┤
│  오늘의 해양 예보 차트    │
│  ┌──┬──┬──┬──┬──┐      │
│  │12│15│18│21│24│      │  ← 3시간 단위
│  │🌊│🌊│🌊│🌊│🌊│      │
│  │📊│📊│📊│📊│📊│      │  ← 스크롤 가능
│  └──┴──┴──┴──┴──┘      │
│                         │
└─────────────────────────┘
│  🏠  📝  📊  ⚙️          │  ← 탭바
└─────────────────────────┘
```

#### 사용자 플로우
1. 앱 실행 → 메인 화면 (마지막 선택 해변)
2. 상단 드롭다운 → 해변 변경
3. 통계 카드 좌우 스와이프 → 다양한 통계 확인
4. 차트 스크롤 → 시간대별 예보 확인
5. 우측 상단 ⓘ → 도움말

## 데이터 구조

### 해변 (Beach)
- 해변 고유 ID
- 지역 (강원도, 제주도 등)
- 해변명
- 위치 (위도, 경도)
- 메모

### 해양 차트 (Chart)
- 해변 ID
- 날짜 및 시간
- 풍향 및 풍속
- 파도 방향, 높이, 주기
- 수온
- 날씨 정보
- 기온

### 서핑 기록 (SurfRecord)
- 기록 고유 ID
- 해변 ID
- 서핑 시작/종료 시간
- 해당 시간대 차트 데이터
- 평점 (1~5점)
- 개인 메모
- 핀 고정 여부

### 해변 목록 (30+ 스팟)

| 지역 | 해변 |
|------|------|
| **양양** | 죽도, 인구, 기사문, 갯마을, 남애3리, 물치, 설악, 동산, 하조대, 동호, 낙산, 정암 |
| **고성/속초** | 송지호, 천진, 속초해변 |
| **강릉/동해/삼척** | 금진, 경포, 대진, 후화 |
| **제주** | 중문, 이호테우, 월정, 사계 |
| **부산** | 송정 A/B/C/D, 다대포 |
| **포항/울산** | 신광만 A/B, 월포, 진하 |
| **서해/남해** | 만리포, 고풍 남열 해돋이 |

## FAQ

### 사용자 관점

**Q: 예보가 실제와 다르면 어떻게 하나요?**
A: 기록 작성 시 실제 조건을 메모로 남길 수 있습니다. 향후 관측치 데이터 추가를 검토하고 있습니다.

**Q: 핸드폰을 바꾸면 기록이 사라지나요?**
A: v1.1에서 iCloud 백업 기능이 추가될 예정입니다.

**Q: 무료인가요?**
A: v1.0은 완전 무료입니다. v2.0부터 프리미엄 기능이 추가될 수 있습니다.

### 운영 관점

**Q: API 비용은 얼마나 드나요?**
A: Open-Meteo는 일 10,000콜까지 무료입니다. Firebase는 사용량에 따라 과금됩니다.

**Q: 서버 장애 시 어떻게 대응하나요?**
A: 로컬 캐시 데이터를 표시하고, 모니터링 시스템으로 즉시 알림을 받습니다.

**Q: 새로운 해변을 추가하려면?**
A: 서버측 locations.json 파일에 위도/경도를 추가하면 자동으로 데이터가 수집됩니다.
