## 프로젝트 개요

### 앱 소개
**DoSurf(두섭이)**는 서퍼들이 서핑 기록을 체계적으로 관리하고, 해양 정보를 통해 서핑 조건을 확인할 수 있도록 돕는 서핑 전문 앱입니다.

### 핵심 가치 제안
1. **한국 특화**: 한국 주요 서핑 스팟 기반 데이터 제공
2. **초보자 친화**: 복잡한 기상 데이터를 시각적으로 단순화
3. **개인화 기록**: 서핑 기록/평점을 통한 회고
4. **멀티 디바이스**: iPhone과 Apple Watch 연동

### 제품 구성
- **플랫폼**: iPhone 앱, Apple Watch 앱, Live Activity 위젯
- **언어**: 한국어

## 타겟 사용자

### Primary Persona: 초보 서퍼
- **니즈**: 오늘 파도 조건 확인, 간단한 서핑 기록
- **페인포인트**: 기상 데이터 해석이 어려움

### Secondary Persona: 중급 서퍼
- **니즈**: 파고/파주기/풍향 상세 확인, 기록 기반 비교
- **페인포인트**: 기록 정리의 번거로움

## 핵심 기능

### 1. 해양 예보 차트
**제공 정보**
- 파도 높이/방향/주기, 풍향/풍속, 수온, 기온, 날씨

**기능**
- 해변별 시간대 차트 리스트 제공
- Pull-to-refresh 수동 새로고침
- 앱 재활성화 시 현재 시간대 근처로 자동 포커싱

### 2. 서핑 기록 관리

**기록 생성 (2가지)**
- **서핑 시작/종료 플로우**: 탭바 중앙 버튼으로 시작/종료 → 기록 화면 이동
- **기록 바로하기**: 시작/종료 없이 기록 화면 진입

**기록 항목**
- 서핑 날짜 및 시작/종료 시간
- 해변 위치
- 해당 시간대 차트 스냅샷 저장
- 개인 평점(1~5점)
- 메모(자동 임시 저장)
- 핀 고정

**기록 수정/삭제**
- 기록 카드의 더보기 메뉴에서 수정/삭제
- 핀 고정/해제 토글

### 3. 기록 조회 및 필터

**기록 리스트**
- 날짜/해변/평점 표시
- 기록별 차트 스냅샷 미리보기

**필터/정렬**
- 필터: 전체, 핀 고정, 해변 선택
- 정렬: 최신순/과거순/평점 높은순/낮은순

**상세 보기**
- 메모 보기(시트 모달)

### 4. 통계 및 분석

**카드 스와이프 구조**
- 통계 카드 영역 좌우 스와이프
- 페이지 인디케이터 제공

**제공 카드**
- 현재 해변 기준 파도/바람 카드
- 최근 기록 차트
- 고정 기록 차트

### 5. 해변 선택

**기능**
- 지역(카테고리)별 해변 목록
- 해변 선택
- 마지막 선택 해변 자동 저장/복원

### 6. Live Activity (잠금화면/다이내믹 아일랜드)

**기능**
- 서핑 시작 시 Live Activity 자동 시작
- 경과 시간 표시
- 서핑 종료 시 Live Activity 종료

### 7. Apple Watch 앱

**기능**
- 서핑 세션 시작/종료
- 거리/시간/심박/칼로리/스트로크 수 표시
- WatchConnectivity로 iPhone에 데이터 전송
- iPhone 수신 확인 화면 제공

### 8. 추가 편의 기능

- 메모 자동 임시 저장/복원
- 서핑 중 상태 지속(앱 재시작 후 복원)

## 화면 설계 및 사용자 플로우

### 1. 메인 화면 (대시보드)

```
┌─────────────────────────┐
│  📍 해변 선택 ▼         │
├─────────────────────────┤
│   통계 카드 스와이프     │
│   ○ ● ○                │
├─────────────────────────┤
│  해양 예보 차트 리스트   │
│  (3시간 단위 스크롤)     │
│  ↓ Pull-to-refresh      │
└─────────────────────────┘
│  탭바 + 중앙 서핑 버튼   │
└─────────────────────────┘
```

**플로우**
1. 앱 실행 → 마지막 선택 해변 로드
2. 해변 변경 → 차트 업데이트
3. 카드 스와이프 → 통계 확인
4. Pull-to-refresh → 새로고침
5. 앱 재활성화 → 현재 시간대 포커싱

### 2. 서핑 기록 시작/종료

**플로우**
1. 중앙 서핑 버튼 탭
2. 서핑 시작/기록 바로하기 선택
3. 서핑 종료 → 기록 화면 이동

### 3. 기록 평가 화면

**플로우**
1. 날짜/시간 확인 및 수정
2. 차트 스냅샷 확인
3. 평점/메모 입력
4. 저장

### 4. 기록 조회 화면

**플로우**
1. 기록 리스트 확인
2. 필터/정렬 선택
3. 더보기 메뉴로 핀/수정/삭제
4. 메모 보기

### 5. 해변 선택 화면

**플로우**
1. 지역 선택
2. 해변 선택
3. 선택 해변 저장 및 적용

### 6. Apple Watch 화면

**플로우**
1. Watch 앱 실행
2. 서핑 세션 시작/종료
3. iPhone으로 데이터 전송

## 데이터 구조

### 해변 (Beach)
- 해변 고유 ID
- 지역 카테고리
- 해변명
- 위치(위도, 경도)

### 해양 차트 (Chart)
- 해변 ID
- 날짜 및 시간
- 풍향/풍속
- 파도 방향/높이/주기
- 수온/기온
- 날씨

### 서핑 기록 (SurfRecord)
- 기록 고유 ID
- 해변 ID
- 서핑 시작/종료 시간
- 차트 스냅샷
- 평점
- 메모
- 핀 고정 여부

### 사용자 설정
- 마지막 선택 해변
- 서핑 중 상태(시작/종료 시간)
- 임시 메모
